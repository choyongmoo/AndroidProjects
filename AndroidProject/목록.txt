약속 목록 페이지

사용자가 참여 중인 약속들을 볼 수 있는 화면.

예: "오늘의 약속", "다가오는 약속", "지난 약속" 등.

약속 생성 페이지

제목, 시간, 장소, 참여자 등을 입력해서 새 약속을 생성하는 화면.

지도를 띄워서 위치 선택 가능하게 하면 좋아.

약속 상세 페이지

약속 정보(장소, 시간, 친구들) 확인.

약속 장소 지도에 표시 + 현재 참여자들의 위치 보여주기.

채팅 기능도 있으면 이 화면에 포함될 수 있음.

지도 공유 페이지 / 실시간 위치 페이지

약속 시간 전후에 서로의 위치가 지도에서 보이는 화면.

GPS 접근 권한 필요.

설정 / 마이페이지

알림 설정, 계정 정보, 로그아웃 등.
