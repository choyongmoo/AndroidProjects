✅ API 목록 (기능별 정리)
🔐 [1. 인증 및 사용자 관리]
기능    메서드    경로    설명
회원가입    POST    /api/auth/signup    사용자 생성
로그인    POST    /api/auth/login    JWT 토큰 발급
사용자 정보 조회    GET    /api/users/me    내 정보 조회
사용자 검색    GET    /api/users/search?query=xxx    닉네임/이메일로 사용자 검색

👥 [2. 친구 및 친구 그룹 관리]
기능    메서드    경로    설명
친구 요청 보내기    POST    /api/friends/request    친구 요청
친구 요청 수락    POST    /api/friends/accept    요청 수락
친구 목록 조회    GET    /api/friends    나의 친구 목록
그룹 생성    POST    /api/groups    친구 그룹 생성
그룹에 친구 추가    POST    /api/groups/:groupId/members    그룹 멤버 추가
그룹 목록 조회    GET    /api/groups    내가 속한 그룹 조회
그룹 상세 조회    GET    /api/groups/:groupId    그룹 정보, 멤버 포함

📍 [3. 장소 즐겨찾기 및 약속 생성]
기능    메서드    경로    설명
그룹 장소 즐겨찾기 등록    POST    /api/groups/:groupId/places    장소 등록
그룹 장소 목록 조회    GET    /api/groups/:groupId/places    즐겨찾기 목록
약속 목록 조회    GET    /api/groups/:groupId/appointments    그룹 약속 리스트

=======현성=======
약속 생성    POST    /api/appointments    시간, 장소 포함
약속 상세 조회    GET    /api/appointments/:id    도착 로그, 장소 등 포함

📡 [4. 위치 공유 및 도착 체크]
기능    메서드    경로    설명
내 위치 업데이트    POST    /api/location/update    위도/경도 업로드
친구 위치 조회    GET    /api/groups/:groupId/locations    그룹 멤버 위치 실시간 확인
약속 도착 체크    POST    /api/appointments/:id/arrive    도착 시간 기록

💸 [5. 벌금 시스템]
기능    메서드    경로    설명
벌금 자동 계산 (서버 트리거)    POST    /api/appointments/:id/calculate-penalty    마지막 도착자 벌금 계산
벌금 내역 조회    GET    /api/penalties/:appointmentId    해당 약속 벌금 분배 내역
🧠 인증 및 권한 처리

    모든 API는 Authorization: Bearer <JWT> 헤더 사용

    그룹, 친구 요청 등에서는 접근 권한 체크 미들웨어 필요

🔄 WebSocket (선택)

    친구 위치 실시간 공유는 WebSocket으로 구현 추천: /ws/location
